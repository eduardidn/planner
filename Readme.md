## Installation

This projects needs the following libraries:

- cmake
- pqxx

To run the project the path to pqxx has to be redefined based on the user device (pkg-config had problems to find it)

### Compilation steps:

From the source folder run:

```
cd build
rm -rf *
cmake ..
make
```
